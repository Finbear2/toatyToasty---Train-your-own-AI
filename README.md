# toatyToast
A small AI model and trainer built on Python and Pytorch


### Why does this exist?
I was bored and wanted to learn AI. So, with a little help of Claude, I coded this along with taking notes on it so I'd better understand it in learning.txt

### Model
I call this toatySpear, it's a 2m parameter model based off of the public works of shakespear. Took about 30 minutes to train but it's a cool kidna proof of conecpt and alsso taught me alot about AI and getting the values
