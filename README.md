# toatyToast
A small AI model and trainer built on Python and Pytorch


### Why does this exist?
I was bored and wanted to learn AI. So, with a little help of Claude, I coded this along with taking notes on it so I'd better understand it in learning.txt

### Model
In this repo there's two models:

**toatySpear V2**, it's a 800k parameter model based off of the public works of shakespear. Took about 30 minutes to train but it's a cool kidna proof of conecpt and also taught me alot about AI and getting the values.

**WindInTheWonderland**, it's a 1m paramerter model tarined off of wind in the willows and Alice in wonderland. Took about 7 minutes to train as I made some speed improvements including optimizer, using GPU and adding a loss target.
